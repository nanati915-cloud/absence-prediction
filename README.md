# 保育園欠席予測・分析システム

### 〜AIによる収束予測と報告書自動作成ツールの開発〜

2021年6月から2026年6月までの5年間にわたる園児の欠席実績データを基に、統計学的なアプローチと機械学習（LightGBM）を用いて、将来の欠席リスクの収束（健康の安定期）を予測し、解析結果を自動でExcelレポート化する統合システムです。

---

## 開発背景

- データの価値化: 5年間、家庭で記録し続けた欠席実績を、単なる記録から「将来予測」という価値へ変換することを試みました。
- 業務改善職への意識: 将来の業務改善職を意識し、複雑な集計・解析・考察、および画像挿入を含む「報告書作成工程」を完全自動化するツールとして構築しました。
- 知識の実践: 「Python3認定データ分析試験」の合格で得た知識を、実務的な特徴量エンジニアリング（平均・分散・トレンド算出等）やモデル構築に反映させました。

## 主な機能

1. 高度なETL処理: 非定型なExcel配置から、月齢・学年・登園年数などの多軸データを正確に抽出・クレンジング。
2. AI未来予測: LightGBMを用いた自己回帰シミュレーションにより、24ヶ月先までの月間欠席日数を推論。
3. 独自判定ロジック: 単月の数値に惑わされず、AIの判定窓（WINDOW）から過去のノイズが消えるまでの「統計的信頼性」を考慮した健康安定期の特定。
4. プロフェッショナル・レポーティング:
   - グラフの自動挿入（アスペクト比維持）
   - 予測数値に基づく考察文の動的組み立て
   - 結合セル対応: Openpyxlの制約を座標計算で回避し、テンプレート内のマージされたセルへの安全な書き込みを実現。
5. ロジック検証用データ生成: 他モジュールの動作検証のため、統計的一貫性を持ったテストデータを生成するシミュレーター（`generate_data.py`）を実装。

テクノロジースタック

- Language: Python 3.13+
- Data Analysis: Pandas, NumPy
- Machine Learning: LightGBM, Scikit-learn
- Visualization: Matplotlib
- Excel Processing: Openpyxl
- Environment: Pathlibによるクロスプラットフォーム対応
